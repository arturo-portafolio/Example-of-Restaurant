# Example of Restaurant

A clean, responsive restaurant landing page demo built with **HTML, CSS, and vanilla JavaScript**. It includes a modern layout (Hero, About, Menu, Schedule, Contact), a **dynamic menu grid rendered from JavaScript**, a **Google Maps embed**, a **WhatsApp contact button**, and a lightweight **floating chatbot UI** for customer interaction.

## Live Demo
https://example-of-restaurant.vercel.app/

## Features
- Responsive navigation (mobile menu toggle)
- Dynamic menu items (loaded via JS into the grid)
- Contact section with address/phone + WhatsApp button
- Embedded Google Maps location
- Floating chatbot bubble + chat window UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Run Locally
Just open `index.html` in your browser (or use any local server like VS Code Live Server).
